Assunto: Parágrafos, Quebra de Linhas e Caracteres Especiais

Para escrever parágrafos, basta colocar o texto entre os pares de marcação. Para iniciar outro parágrafo, repete o mesmo procedimento.
Exemplo: <p> texto aqui </p>

Em caso de quebra de linha, deve usar apenas o sinalizador <br>

Para utilizar emoji, deve-se pesquisar o seu código em hexadecimal e substituir o U+ por $#x
Exemplo: U+1F600 torna-se: 😀ao trocar U+ por $#x e manter o restante.

Para colocar uma linha horizontal, usa-se apenas <hr>

Outro detalhe é que para fazer um comentário no código, deve-se fazer: <!-- colocar o comentário nesse espaço -->. Cuidado para não esquecer os traços.
